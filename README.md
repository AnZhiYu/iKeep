# iKeep
iKeep：基于uni-app和云开发的时间投资微信小程序

> iKeep时间投资是一个帮助使用者规划、记录自己时间投入的小程序。可以记录看书多少小时、学习多少小时、弹琴多少小时等需要时间积累才能提高的技能。对任何一件事情投入 1 万小时，你都可以成为这个领域的专家。
